# Rising Waters: A Machine Learning Approach to Flood Prediction

## Project Overview
This project predicts the possibility of floods using Machine Learning based on weather and rainfall data. A Flask web application is used to provide predictions through a user-friendly interface.

## Technologies Used
- Python
- Pandas
- NumPy
- Scikit-learn
- Flask
- Matplotlib
- Seaborn
- Joblib

## Project Structure
- app.py
- train_model.py
- dataset/
- models/
- templates/
- static/

## How to Run
1. Install the required libraries.
2. Run train_model.py to train the model.
3. Run app.py.
4. Open http://127.0.0.1:5000 in your browser.

## Output
The application predicts whether a flood is likely to occur based on the given input values.
